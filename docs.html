Getting started

To get started, you'll first need to generate your own code snippet. Using only your email address, we'll create you your own, unique code block; just like if you were installing Google Analytics.

When you've got your code snippet, you'll need to paste it into the same HTML file as your form; it doesn't matter where, as long as it's in the same form.
