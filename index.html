Howdy

Howdy detects forms on your web page, automatically validates them and sends submissions straight to your chosen email address

Enter your email address to generate your Howdy code snippet:

Copy & paste this snippet onto your form page and try it out.

              

                
<script src="?token=your-email@example.com" data-howdy-recipient=""></script>

Boring Notes

  • Howdy will only include the contents of form fields with a name attribute.
  • You can reuse your snippet for as many email inboxes as you like, just swap the data-howdy-recipient attribute for each new inbox.

Highly Customiseable

Add & override validations by adding HTML attributes

Add these to your inputs and textareas to add or override default validations:

  • data-howdy-validate=“none”// No validation on this input
  • data-howdy-validate=“required”// Some value is required
  • data-howdy-validate=“email”// Email address is required
  • data-howdy-validate=“number”// A number is required
  • data-howdy-validate=“date”// A date is required
  • data-howdy-validate=“tel”// No letters are allowed

Send emails to a new recipient under the same token:

You can use a single Howdy token across multiple email addresses by simply adjusting the recipient on your script tag. There's no need to generate a new token:

  • <script src=“…“ data-howdy-recipient=“another@email.com“></script>

Customise your after send message (advanced):

With a little bit of JavaScript knowledge, you can override the default alert message for Howdy:

  • <script type="text/javascript">
  •   window.howdyEvents = {
  •     afterSend: function(messageData) {
  •       alert("My custom message!");
  •     }
  •   };
  • </script>
