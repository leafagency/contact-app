Custom Validation

Override default validation by adding a data-howdy-validate attribute to your input

  • data-howdy-validate=“none” // No validation on this input
  • data-howdy-validate=“required” // Some value is required
  • data-howdy-validate=“email” // An email address in required
  • data-howdy-validate=“number” // A number is required
  • data-howdy-validate=“noLetters” // No letters allowed in the input
