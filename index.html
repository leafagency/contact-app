How do I use it?

First your email to generate your code snippet

Then, just copy this code snippet onto your page:

<script type="text/javascript" src="http://howdy-api-staging.herokuapp.com/static/howdyClient.js?email=your-email-here@example.com"></script>

That's it, Howdy will take care of the rest for you.

  • name
  • first_name

  • lastName

Specifying your own validations

You can override any default validations or add your own custom fields by adding a data-howdy-validate="" attribute to your input.

Supported validations are as follows:

  • none - No validation on this input
  • required - Some value is required
  • email - An email address is required
  • number - A number is required
  • noLetters - No letters allowed in the input
